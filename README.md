
<h1 align="center"> Delivery App  </h1>

#### This iOS application is a simple delivery app built with MVVM architecture. It uses Alamofire for networking and RxSwift for reactive programming. 

## 🍔 🍟 Technologies Used
- Alamofire
- MVVM
- UIKit
- Alamofire
- RxSwift
- MapKit
- Notification
